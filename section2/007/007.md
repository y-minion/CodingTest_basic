## 문제

봉우리
지도 정보가 N\*N 격자판에 주어집니다. 각 격자에는 그 지역의 높이가 쓰여있습니다. 각 격자
판의 숫자 중 자신의 상하좌우 숫자보다 큰 숫자는 봉우리 지역입니다. 봉우리 지역이 몇 개
있는 지 알아내는 프로그램을 작성하세요.
격자의 가장자리는 0으로 초기화 되었다고 가정한다.
만약 N=5 이고, 격자판의 숫자가 다음과 같다면 봉우리의 개수는 10개입니다.
▣ 입력설명
첫 줄에 자연수 N이 주어진다.(1<=N<=50)
두 번째 줄부터 N줄에 걸쳐 각 줄에 N개의 자연수가 주어진다. 각 자연수는 100을 넘지 않는
다.
▣ 출력설명
봉우리의 개수를 출력하세요.
▣ 입력예제 1
5
5 3 7 2 3
3 7 1 6 1
7 2 5 3 4
4 3 6 4 1
8 7 3 5 2
▣ 출력예제 1
10

## 1차 풀이

가장자리에 0으로 이루어진 새로운 배열 삽입후 진행

```javascript
function solution(arr) {
  arr.forEach((el) => {
    el.push(0);
    el.unshift(0);
  });
  arr.push(Array.from({ length: arr.length + 2 }, () => 0));
  arr.unshift(Array.from({ length: arr.length + 2 }, () => 0));

  let count = 0;
  for (let i = 0; i < arr.length; i++) {
    for (let j = 0; j < arr.length; j++) {
      if (arr[i][j] === 0) continue;
      const top = arr[i - 1][j];
      const bottom = arr[i + 1][j];
      const right = arr[i][j + 1];
      const left = arr[i][j - 1];
      const cur = arr[i][j];
      if (top < cur && bottom < cur && right < cur && left < cur) {
        count++;
      }
    }
  }
  return count;
}
```

## 2차 풀이

방향벡터 도입해 벡터 배열 순환하며 풀이

```javascript
function solution(arr) {
  const dx = [0, 0, 1, -1];
  const dy = [-1, 1, 0, 0];
  let count = 0;

  for (let i = 0; i < arr.length; i++) {
    for (let j = 0; j < arr.length; j++) {
      let flag = 0;
      const cur = arr[i][j];

      dx.forEach((x, idx) => {
        const y = dy[idx];
        if (
          i + y < 0 ||
          i + x < 0 ||
          arr.length - 1 < i + x ||
          arr.length - 1 < i + y
        )
          return;
        const other = arr[i + y][j + x];
        if (other >= cur) flag = 1;
      });
      if (flag === 0) count++;
    }
  }

  return count;
}
```
