## 문제

큰 수 출력하기
N(1<=N<=100)개의 정수를 입력받아, 자신의 바로 앞 수보다 큰 수만 출력하는 프로그램을 작
성하세요.(첫 번째 수는 무조건 출력한다)
▣ 입력설명
첫 줄에 자연수 N이 주어지고, 그 다음 줄에 N개의 정수가 입력된다.
▣ 출력설명
자신의 바로 앞 수보다 큰 수만 한 줄로 출력한다.
▣ 입력예제 1
6
7 3 9 5 6 12
▣ 출력예제 1
7 9 6 12

## 1차 풀이

큰수 출력 -> 필터값
입력 배열 -> 큰 수 결과 배열

1. 첫번째 무조건 포함
2. 배열 탐색
3. 이전 요소와 현재 요소 비교
   - 이때 제일 루프때 flag 등록
   - 그리고 매 루프의 마지막에 flag 업데이트

```javascript
function solution(arr) {
  let flag;
  return arr.filter((num, idx) => {
    let result = false;
    if (idx === 0) {
      flag = num;
      return true;
    }
    if (flag < num) {
      result = true;
    }
    flag = num;
    return result;
  });
}
```

## 2차 풀이

굳이 flag까지 사용하지 않고, 현재 idx의 -1 번째 요소와 비교하면 된다.

```javascript
function solution(arr) {
  return arr.filter((num, idx) => {
    if (idx === 0) return true;
    if (arr[idx - 1] < num) return true;
  });
}
```
