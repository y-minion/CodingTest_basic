## 문제

문자열 압축
알파벳 대문자로 이루어진 문자열을 입력받아 같은 문자가 연속으로 반복되는 경우 반복되는
문자 바로 오른쪽에 반복 횟수를 표기하는 방법으로 문자열을 압축하는 프로그램을 작성하시
오. 단 반복횟수가 1인 경우 생략합니다.
▣ 입력설명
첫 줄에 문자열이 주어진다. 문자열의 길이는 100을 넘지 않는다.
▣ 출력설명
첫 줄에 압축된 문자열을 출력한다.
▣ 입력예제 1
KKHSSSSSSSE
▣ 출력예제 1
K2HS7E

## 1차 설계

```javascript
function solution(s) {
  const arr = [...s];
  let res = "";
  arr.forEach((char, i) => {
    if (char === arr[i - 1]) return;
    let count = 0;
    for (j = i; j < arr.length; j++) {
      if (char !== arr[j]) {
        /*
            아래 로직이 원래 여기에 있어서 계속 버그 발생...
              count = count === 1 ? "" : count;
              res += `${char}${count}`;

              */
        break;
      }

      count++;
    }
    count = count === 1 ? "" : count;
    res += `${char}${count}`;
  });
  return res;
}
```

## 2차 풀이

1-pass로 진행하며 루프마다 직전 요소와 값을 비교한다
이때 동일하면 count를 증가시키고, 다르다면 직전의 카운트가 2 이상인 경우에만 카운트를 더해준다.
그리고 현재 요소를 추가하고 count를 1로 초기화 한다.

엣지케이스는 마지막 요소이다. 이때 동시에 연속된 숫자(count 1이상) 인 경우에만 다시 요소를 추가해줘야한다.

```javascript
function solution(str) {
  let count = 0;
  const res = [];
  [...str].forEach((el, idx, arr) => {
    if (arr[idx - 1] !== el) {
      if (count > 1) res.push(count); //전의 횟수 추가

      res.push(el);
      count = 1;
      return;
    }
    count++;
    if (idx === arr.length - 1 && count > 1) {
      res.push(count);
    }
  });

  return res.join("");
}
```
