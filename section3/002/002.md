## 문제

숫자만 추출
문자와 숫자가 섞여있는 문자열이 주어지면 그 중 숫자만 추출하여 그 순서대로 자연수를 만
듭니다.
만약 “tge0a1h205er
”에서 숫자만 추출하면 0, 1, 2, 0, 5이고 이것을 자연수를 만들면 1205
이 됩니다.
추출하여 만들어지는 자연수는 100,000,000을 넘지 않습니다.
▣ 입력설명
첫 줄에 숫자가 썩인 문자열이 주어집니다. 문자열의 길이는 50을 넘지 않습니다.
▣ 출력설명
첫 줄에 자연수를 출력합니다.
▣ 입력예제 1
g0en2T0s8eSoft
▣ 출력예제 1
208

## 풀이

정규표현식을 이용해 숫자만 추출한뒤, join을 통해 숫자를 합친다.

```javascript
<html>
  <head>
    <meta charset="UTF-8" />
    <title>숫자만 추출</title>
  </head>
  <body>
    <script>
      function solution(s) {
        //숫자만 추출 -> 정규표현식 사용해 math 메서드 사용
        const regex = /\d+/g;
        const transform = s.match(regex);
        if (transform[0] === "0") transform.shift(); //굳이 제일 앞의 0은 제거할 필요가 없다. JS의 Number가 알아서 10진수로 판단해준다.
        return Number(transform.join(""));
      }

      let str = "g0en2T0s8eSoft";
      console.log(solution(str));
    </script>
  </body>
</html>

```

## 2차 풀이(개선)\_replace 사용

기존에 숫자를 추출해서 join하는 과정을 대신 replace를 사용해 숫자가 아닌 나머지 요소들을 공백으로 대체한 문자열을 반환한다.

```javascript
function solution(s) {
  //숫자를 추출해 join하는 과정 없이 **숫자가 아닌** 문자열들은 공백으로 제거한다.
  const regex = /\D+/g;
  const replaced = s.replace(regex, "");
  return Number(replaced);
}
```
