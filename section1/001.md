## 001문제

100이하의 자연수 a,b,c를 입력 받는다.
입력받은 세 수 중 **_가장 작은 값_**을 출력해라.
이때 정렬을 사용하면 안된다.

## 설계

입력받은 매개변수들을 모두 크기 비교한다.
매개변수를 배열에 넣자.
각 하나의 요소를 선택해서 다른 요소들과 크기를 비교해야한다.

## 풀이

```javascript
function solution(a, b, c) {
  let answer;
  if (a < b) {
    answer = a;
  } else {
    answer = b;
  }

  if (c < answer) answer = c;

  return answer;
}

console.log(solution(2, 5, 14));
```

## 의문

이때 만약 매개변수가 100개라면? -> 위의 방식으로는 절대 해결할 수 없다.

## 설계2

1. 함수의 배개변수를 모두 배열로 받는다.
2. 매개변수의 배열의 요소 한개를 선택한다. -> 해당 요소를 제외한 나머지 요소들의 크기 비교 실행.
3. 현재 요소 < 비교 요소 -> 비교요소 삭제(0으로 표식을 남긴다)
4. 현재 요소 > 비교 요소 || 현재 요소 === 비교 요소 -> 현재 요소 삭제(=0으로 표식 남긴다.) + 다음 루프로 점프

```javascript
function solution(...arr) {
  let answer;

  //하나씩 요소 선택
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === 0) continue; // 0이라는 숫자를 요소 삭제 대신 표시 용도로 사용한다
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[j] === 0) continue;
      if (arr[i] < arr[j]) {
        arr[j] = 0;
      } else {
        arr[i] = 0;
        break;
      }
    }
  }
  arr.forEach((el) => {
    if (el === 0) return; //0으로 표식 처리한 요소를 제외한 남은 최소값 찾기
    answer = el;
  });
  return answer;
}

console.log(solution(6, 5, 14, 44, 3, 5, 43, 5, 235));
```
